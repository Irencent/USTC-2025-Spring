B站网红UP主是如何炼成的？

狗熊会

  案例背景：“bilibili”是国内知名的视频弹幕网站，类别覆盖鬼畜、游戏、动漫、番剧等数十个品类。B站拥有粉丝近千万的超级网红，网红UP主的视频点击量动辄上百万，同时也存在大量点击量平平的底层UP主。我们希望通过数据分析找到不同UP主之间热度差异的影响因素。更多案例背景介绍，可以参考B站网红UP主是如何炼成的。

  数据介绍：案例数据提供了B站8360个UP主的数据观测，共20个变量，其中包括UP主id，粉丝数，性别，发表视频数，发表代表作个数，发表相册数，发表文章数，发表频道数，近期视频平均时长，近期视频平均播放量，关联第三方平台信息，个人标签等。

这个案例的学习目标包括：

  1. 了解B站数据；

  2. 对文本数据进行简单处理和描述分析；

  3. 学习使用ggplot2进行绘图；

draw = summarise(group_by(dat,video_tag_combine, red),n = n())# 分品类计算不同网红程度UP主数

  解读：人数上来看，游戏类UP主数量最多，生活类次之。对比不同品类来看，生活、游戏类UP主中大网红人数最多，娱乐类中最少。从各个品类的内部结构来看，游戏类UP主中小透明占比最大，时尚类中UP主小网红占比最大，生活、舞蹈类UP主中大小网红占比基本持平。如果我们对比一下游戏类和生活类，可以发现，作为B站目前流量最大、UP主最集中的两个品类，它们在UP主的分布结构上有着明显的差异：游戏类呈现出“鱼龙混杂”的态势，小透明和大网红都占有相当的比例，更多普通的UP主在游戏区活动；而生活类则更加“高质量”，大部分为具有一定实力，积累了一定人气的网红UP主，并且大网红UP主比例相当可观。

任务三

  将近期视频时长(time_ave20)的单位转为分钟，并用ifelse()函数划分时长区间(time_group)：0至5分钟，5至15分钟，15至30分钟，30至60分钟，60分钟以上。分组计算人数和占比后，绘制不同时长各网红程度的人数占比分布图，并进行简要解读。

dat$time_ave20 = dat$time_ave20/60                          # 将时长转为分钟
dat$time_group = ifelse(dat$time_ave20 <5, "0-5",           # 划分时长区间
                    ifelse(dat$time_ave20 <15,"5-15",
                           ifelse(dat$time_ave20 <30,"15-30",
                                  ifelse(dat$time_ave20 <60,"30-60","60+"))))
# 重新排序时长区间
dat$time_group = factor(dat$time_group,levels = c("0-5","5-15","15-30","30-60","60+"))
draw = summarise(group_by(dat, red, time_group),n = n())     # 分区间计算不同网红程度UP主数
draw$all_n = rep(table(dat$red),each = 5)                    # 计算各网红程度的UP主总数
draw$ratio = draw$n / draw$all_n                             # 计算不同网红程度的各区间占比
ggplot(data = draw,aes(x = time_group, y = ratio, fill = red)) +    # 非堆叠直方图
  geom_bar(stat = "identity", alpha = 0.8, position = "dodge")+
  scale_fill_manual(values = c("#c5c5c5","#1fafe6","#f48aa0"))+ 
  scale_y_continuous(breaks = seq(0,0.6,0.25))+              # 调整Y轴分隔
  xlab("近期视频平均时长(分钟)") + ylab("人数占比")+
  theme_bw() + theme(legend.title = element_blank(),
                     legend.position = "top")                # 在上方添加图例，并去掉图例的标题

  解读：对视频时间，如果以15分钟作为分界线把UP主分为“短视频类”和“长视频类”，可以明显发现B站UP主们以“短视频类”居多。并且对“长视频类”UP主，可以看到大网红的比例相较短视频要明显更低。在B站上如果发布视频时间较长，恐怕会令观众“望而却步”，难以在众多UP主的竞争中得到关注，并且成长为大网红的难度也更大。反过来聚焦短视频为主的UP主们，可以发现超过50%的大网红近期视频平均时长在5~15分钟内，这也可以为新人UP主提供一个参考：视频的时间最好控制在这个区间，在B站获得关注的可能更大。

  解读：与第三方平台关联越多，UP主自我宣传、曝光的渠道就越多，越有可能从各方平台吸收粉丝。在各个第三方平台中，关联微博是大部分UP主的首选。微博作为国内在线社交媒体的头部平台，用户量大、用户活跃度高、信息传播速度快，的确是UP主吸粉的优质渠道。同时，通过考察微博之外的其他关联平台，可以发现不同品类的UP主存在不同的侧重，如：动画、舞蹈、鬼畜和游戏等娱乐类UP主更多使用QQ；生活、影视、科技和时尚这种偏日常生活类的更多考虑微信；而时尚、科技类有带货需求的则有更多UP主关联了小红书和淘宝为变现做好准备。
